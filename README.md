## ✨ 特性
- ✅ 使用 Vue.js 全家桶开发
- 🔴 网易云账号登录（扫码）
- 📺 支持 MV 播放
- 📃 支持歌词显示
- 📻 每日推荐歌曲
- 🚫🤝 无任何社交功能
- 🛠 更多特性开发中

## 📦️ 安装

Electron 版本目前支持 Windows

访问本项目的 [Releases](https://github.com/qier222/YesPlayMusic/releases)
页面下载安装包。

## 灵感来源
